API Biblioteca STL

La API Biblioteca STL permite a los usuarios interactuar con la plataforma para gestionar archivos y proyectos relacionados con modelos 3D STL.

URL Base

Debido a que no se utiliza la reescritura de URL, todos los endpoints se acceden a través de la URL base, seguido de los parámetros GET controlador, metodo y cualquier otro parámetro adicional necesario.

https://fernandofbamzn.duckdns.org/ApiBibliotecaSTL/api.php

Controladores y Métodos

Cada categoria, corresponde a un controlador, por ejemplo Archivos -> ArchivosController
